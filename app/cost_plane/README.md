# Cost Plane Architecture

Neden cost plane gerektiği: Costs -> attribution/allocation -> budgets/guardrails -> unit economics -> variance/forecast -> trust akışını sağlamak için.
Why total spend != useful economics: Toplam harcama verimli kullanıldığını garanti etmez.
Why no hidden subsidy / no cheapness theater: Gizli sübvansiyonlar engellenmeli, ucuz görünen riskler (cheap but fragile) engellenmeli.
Bu fazın sınırları: Gerçek vendor entegrasyonu yok, governance ve attribution var.
