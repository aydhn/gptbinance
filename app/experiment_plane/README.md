# Experiment Plane and Variant Governance

This plane provides canonical governance for all experiments in the platform.

## Objective
To ensure that all research, strategy, and model variants are evaluated with strict baseline discipline, fair comparison contracts, and governed stopping rules.

## Flow
Objective -> Arms -> Exposure Allocation -> Fair Comparison -> Stopping Rules -> Recommendation

## Anti-Patterns
- Hidden traffic shift
- Hidden baseline drift
- Incomparable arms presented as comparable
- Overclaiming "winner" based on weak evidence
