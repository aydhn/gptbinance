# Performance Plane / Benchmark Governance

## Purpose
The performance plane is responsible for answering why a return occurred and how reliable that performance is. It governs:
1. Returns -> Realized / Unrealized / Equity linked returns
2. Benchmarks -> Typed benchmark selection, no hidden switching
3. Attribution -> Selection / Timing / Allocation / Execution / Risk-block separation
4. Drags -> Slippage, funding, fee, carry, idle-cash friction
5. Opportunity -> Captured vs foregone, clipped, blocked (with strict counterfactual caveats)
6. Trust & Equivalence -> Replay / paper / live performance divergence and trusted performance verdicts

## Anti-Goals
- No vanity tearsheets.
- No cherry-picking benchmarks.
- No treating counterfactual or missed profits as guaranteed realized profits.
- No hiding execution/funding drag within signal failures.
