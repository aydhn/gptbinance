# Workspace Manifest, Qualification ve Release Scope Politikası

Her Workspace'in bir manifestosu vardır. Bu manifesto, çalışma alanının durumunu özetler.

## Manifesto İçeriği
- Workspace ID ve profiller.
- Release refs, data governance refs, qualification refs.

Stale (eski/geçersiz) manifestler sistemin güvenlik duruşunu bozacağı için uyarı üretir ve Qualification süreçlerinde Block sebebi olabilir. Scope temizliği (Cleanliness) zorunludur.
