# Migration Debt, Verification ve Release Governance Politikası

Bu politika, birikmiş migration borçlarının (debt) yönetimine odaklanır.

- **Pending Migrations & Debt:** Tamamlanmayan, atlanan veya başarısız olan migration'lar "debt" (borç) olarak kaydedilir.
- **Verification:** Migration sonrası doğrulama başarısız olursa, severity artırılarak release/qualification blocklanabilir.
- **Governance:** Teknik olarak hazır görünen ama migration debt seviyesi yüksek olan sürümler "block" edilebilir.
