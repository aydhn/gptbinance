# Phase 50: Release Readiness Board and Evidence Court Architecture

This document describes the core architecture of the Readiness Board.

- **Candidate Registry**: Tracks promotion candidates.
- **Freeze Snapshot**: Freezes a candidate's context for review.
- **Evidence Court**: Filters admissible vs inadmissible evidence.
- **Readiness Domains**: Aggregates verdicts across domains like Policy, Market Truth, etc.

Direct live activation is strictly prohibited in this phase.
