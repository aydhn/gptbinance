# Incident Command System ve Containment Governance Mimarisi

## Amac
Sistem uzerinde birbirinden bagimsiz olarak calisan (Market Truth, Shadow State, vb) uyari mekanizmalarini (alertleri) merkezi bir framework (Incident Command) altinda toplamak.

## Neden Otomatik 'Self-Healing' Yok?
Incident Command katmani operasyonel 'awareness' ve 'containment' hedefler. Keskin ve otomatik kararlar ile tum sistemin restart edilmesi daha buyuk state drift'lerine yol acabilir.
