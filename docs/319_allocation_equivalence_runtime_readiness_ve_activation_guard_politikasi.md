# Allocation Equivalence, Runtime Readiness & Activation Guard

- Equivalence: Çevrimdışı/replay allocation'lar ile runtime arasındaki uyum test edilir.
- Activation Guard: Uyumsuzluk (Diverged Critical) anında sistem aktivasyonu engeller.
- Trust: Sinyal, bütçe ve eşdeğerlik raporları sentezlenerek bir `Trust Verdict` oluşturulur.
