# Bundle Pinning ve Environment Compatibility Politikası

Tüm bundle'lar; strategy, model, feature, config manifest'leri için strict versiyon pinleri içermelidir.
Eksik pinler bir bloklayıcıdır (blocker).

Release olmadan (release-less) herhangi bir aktivasyon kesinlikle yasaktır.
Environment uyuşmazlığı tespit edildiğinde canlıya alma engellenir.
