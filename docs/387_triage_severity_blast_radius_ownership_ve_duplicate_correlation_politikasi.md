# Triage, Severity, Blast Radius, and Ownership Policy

## Rules
- Triage must strictly separate provisional facts from hypotheses.
- Severity/Urgency changes must have explicit proof notes.
- If blast radius is unknown, it defaults to a global caution state.
- No incident can be unowned. Primary owner is required.
- Duplicate and correlated incidents must be logically linked, not silently merged.
