# Causal Chain, Contributor Taxonomy ve False Root Cause Politikası

- **Proximate vs Root Causes**: Proximate cause tetikleyiciyi işaret eder, root cause ise asıl tasarım/süreç/governance eksikliğini gösterir.
- **Contributor Taxonomy**: Automation gap, missing gate, hidden override, environment drift gibi kategorilerle soruna katkı sağlayan faktörler tanımlanır.
- **False Root Cause**: Sadece bir kişinin hatası diyerek kapatılan, process veya teknik eksiklikleri görmezden gelen RCA'lar reddedilir.
