# SBOM, License, Origin, Runtime Lineage Politikası

- SBOM kayıtları güncel olmalı, eski (stale) SBOM'lar borç (debt) olarak kaydedilmelidir.
- Third-party origin'ler güven derecesine göre incelenmelidir.
- Runtime'da çalışan artifact, release bundle ile eşleşmek zorundadır.
- Drift (Dependency, Build, Runtime, License) tolere edilmez, kaydedilir ve blast radius belirlenir.
