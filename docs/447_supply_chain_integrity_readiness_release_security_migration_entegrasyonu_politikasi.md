# Integration Politikası

Supply Chain Plane diğer katmanları besler:
- Readiness: supply_chain_integrity domain'i release'i engeller.
- Security: unsigned artifact veya unreviewed origin security riskidir.
- Compliance: SBOM ve provenance yasal zorunluluklardır.
- Policy: Belirsiz supply-chain durumları policy_deny üretir.
