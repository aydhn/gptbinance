# Isolation, Tenancy, Secret Scope, Data Scope, Network Scope ve Contamination Politikası

İzolasyon, sadece ayrı isimlere sahip olmak değildir. Gerçek network, state ve compute izolasyonunu gerektirir.
Sırlar (Secrets) "live" ve "non-live" olarak açıkça ayrılmalı, yanlış ortamda gerçek veri veya canlı sırlar kullanılmamalıdır. Çapraz ortam (cross-environment) sızıntıları "contamination" olarak işaretlenir ve trust skorunu engeller.
