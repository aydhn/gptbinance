# Autonomy Integrity Readiness, Constitution, Change, Release, Security Entegrasyonu Politikası

- **autonomy_integrity domain**: Otonom süreçlerin uygunluğunun denetlendiği domain.
- **Integrations**: Constitution, change, release, security, federation ve state sistemleriyle entegrasyonlar.
- **Policy obligations**: Otonom işlemlerde uyulması gereken zorunlu kurallar.
- **Blocker/caution semantics**: Otonomi süreçlerinde engelleyici ve uyarıcı durumlar.
