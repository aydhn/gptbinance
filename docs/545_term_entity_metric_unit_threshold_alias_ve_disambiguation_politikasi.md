# Semantic Plane: Policies for Core Ontology

## Canonical Objects
- **Terms & Entities:** Must have authoritative definitions and clear boundaries.
- **Metrics:** Formulas must have an unambiguous basis.
- **Units & Thresholds:** Cannot be decoupled from operational meaning.
- **Aliases:** A documented alias is NOT equivalent semantics. Aliases must be explicitly classified (safe, dangerous).

## Disambiguation
Where ambiguity exists, disambiguation records must clarify operational intent before Trust Verdicts can be issued.
