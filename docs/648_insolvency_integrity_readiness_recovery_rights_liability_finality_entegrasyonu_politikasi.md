# Insolvency Integrity and Cross-Plane Integration Policy

## Integrity Checks
The Insolvency Plane acts as a truth substrate for the rest of the system:
- **Recovery Plane**: Finality is blocked if the estate contains unresolved class deficits.
- **Rights Plane**: Beneficiary impairment dictates challenge rights.
- **Liability Plane**: Retained exposure is explicitly evaluated post-haircut.
- **Finality Plane**: A final label cannot be applied to a closure associated with a live insolvency contest.

## Readiness Domain
The `insolvency_integrity` domain assesses:
- Estate Clarity
- Claim Rigor
- Priority Discipline
- Plan Integrity
- Residual Deficit Visibility

Any severe warning blocks system progression (e.g., releasing a new product that carries a restitution debt from a prior insolvency).
