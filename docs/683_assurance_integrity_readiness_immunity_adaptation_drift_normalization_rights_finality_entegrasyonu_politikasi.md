# Assurance Integrity & Cross-Plane Integrations

- **Integrations**: Immunity, adaptation, drift, normalization, recovery, rights, liability, and finality operations MUST reference current assurance objects.
- **Readiness Domain**: An entire readiness domain focuses on Assurance Integrity.
- **Policy Operations**: Insufficient evidence, missing certifications, or expired attestations block higher-tier actions.
- **Review Packs**: Collections of claims/evidence used in reviews and readouts.
